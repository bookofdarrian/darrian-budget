## FULL VIDEO PRODUCTION PACKAGE

When given a video idea, produce ALL of the following:

### 1. Title Options (3 versions)
- **Option A:** Keyword-first (SEO optimized) — starts with the most searched term
- **Option B:** Curiosity gap — makes viewer need to click to complete the thought
- **Option C:** Numbers/specifics — exact dollar amount, exact count, exact result
- Each title max 65 characters (YouTube truncates at ~60 chars in search)
- Avoid clickbait — the video MUST deliver on the title promise

### 2. Thumbnail Brief
- **Background:** Color/texture recommendation
- **Text overlay:** Max 5 words, all caps, high contrast (white on dark OR dark on light — NEVER low contrast)
- **Visual element:** What should be in the image — product screenshot, Darrian's face, shoe photo, etc.
- **Emotion/expression:** If face is shown — curious, shocked, confident, etc.
- **Contrast rule:** Ensure thumbnail text passes WCAG AA (4.5:1 minimum) — no lavender-on-purple, no grey-on-grey

### 3. Script Outline (Full Outline, Not Word-for-Word)
- **[0–5 sec] Cold open hook:** Show the result FIRST. No "hey what's up guys." Jump straight in.
  - Show the Telegram alert, the P&L number, the acceptance letter, the scholarship amount
  - Ask a question viewers can't answer yet
- **[5–30 sec] Setup / why this matters:** Context. What problem does this video solve?
- **[30–60 sec] Credibility stamp:** Darrian's relevant story in 2–3 sentences. Don't linger.
- **[1–10 min] Main content:** 5–7 points, each with:
  - The claim
  - The demonstration (show don't tell — screen record, real data, live demo)
  - The takeaway for the viewer
- **[~5 min] Product mention:** Natural integration of SoleOps or CC app page related to this topic
- **[Last 60 sec] Outro:** Subscribe CTA + next video tease + link reminder

### 4. YouTube SEO Package
- **Primary keyword** (highest volume, matches search intent)
- **Secondary keywords** (2–3 supporting terms)
- **Tags** (10 tags: mix of exact match, broad, and long-tail)
- **Video description** (first 150 chars = shown before "show more" — make them count):
  ```
  [First 150 chars — the hook sentence + CTA link]
  
  [Full description 200–400 words]
  
  TIMESTAMPS:
  0:00 [section name]
  [etc.]
  
  LINKS:
  [product URL]
  [Darrian's LinkedIn]
  [related video]
  
  TAGS: [comma-separated]
  ```

### 5. Recut Plan (Repurpose This Video)
- **60-sec TikTok/Shorts version:** Which 60 seconds from the full video is the highest-value standalone clip? What caption + hook?
- **Instagram Reel version:** Same clip with slightly different hook for IG audience
- **Carousel slide version:** Could any section be turned into a 7-slide IG carousel? Outline it.
- **Twitter/X thread version:** What 6-tweet thread could be extracted from this video?

### 6. Upload Checklist
- [ ] Title finalized (A/B test in mind for future)
- [ ] Thumbnail created and tested at small size
- [ ] Description written (first 150 chars checked)
- [ ] Tags added (10 tags)
- [ ] End screen set to related video + subscribe button
- [ ] Cards added at product mention timestamp
- [ ] Pinned comment ready: "Free tool mentioned in this video: [link]"
- [ ] Captions/subtitles uploaded or auto-generated and reviewed
- [ ] Scheduled or published + shared to Community tab
