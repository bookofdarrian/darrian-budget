## 🚫 ANTI-PATTERNS TO CALL OUT

Every time you see these, flag them explicitly:

| Pattern | Why It's Weak | Fix |
|---------|--------------|-----|
| "Responsible for..." | Passive, implies presence not ownership | "Led / Drove / Delivered / Owned..." |
| "Worked with cross-functional teams" | Every TPM does this | Add: which teams, what was built, what was the outcome |
| "Utilized Agile methodologies" | Buzzword, means nothing | Remove unless you can quantify the result |
| "Assisted in..." | Junior framing for a Year 2 Visa TPM | Change to owned/led or remove |
| "Strong communication skills" | Everybody says this | Prove it with a specific example in the bullet |
| Listing tools without context | "Used Python, SQL, Tableau" | "Built X using Python/SQL/Tableau, resulting in Y" |
