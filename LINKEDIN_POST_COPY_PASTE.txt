I built a system that ships production code while I sleep.

I run a personal finance platform, sneaker resale op, and college prep product — on top of my day job at Visa. More ideas than hours. The bottleneck was shipping.

I took the gated SDLC I learned at Visa and gave it to AI agents.

feature → dev → qa → staging → main (prod)

6 agents run every night at 11PM on my home lab:
🧠 Planner → ⚙️ Backend → 🎨 UI → 🧪 Tests → ✅ QA → 🚀 Git Bot → PR

I wake up, review the PR, and merge. That's it.

What makes it different: guardrails are embedded in the system — not repeated every session. It knows my codebase. If tests fail, nothing commits.

73+ pages shipped · ~$1/night (Claude Opus 4) · $0/month infra

The pipeline is the product. Guardrails aren't bureaucracy — they're trust.

The system doesn't replace my judgment. It amplifies it. A human still drives. The AI handles the road.
